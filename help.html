Login Screen

In the login screen, enter the username and password of your account.

If you have not entered this information correctly, you will be prompted to enter valid information.

Main Screen

The main screen provides access to the Employee Database section, the Visitor Attendance, and the Employee Scheduling section.

Click one of these buttons to go to the respective section.

Employee Database

View Employees

This screen provides a detailed view of employee data.

Click "Add" to add new employee.

Click an employee's row in the table and click "Edit" to edit their data.

Add Employee

With this screen, you can add new employees by filling in each field.

Edit Employee

This screen allows you to edit an employee's data easily by changing fields.

Visitor Attendance

View Visitor Attendance

This screen provides an easy way to keep track of visitor attendance.

Click "+1 Adult" if an adult comes in and "+1 Child" when a child comes in.

Select a date in the date picker and click "Generate" to generate the attendance graph for that day.

Attendance Graphs

Once you have selected a date and clicked "generate", if valid attendance data is present, a graph of attendance will appear.

Print this graph by clicking "Print".

Employee Schedules

View Employee Schedules

This screen allows you to edit employee's schedules.

Select a date in the date picker and click the "Go" button to view schedules for that date.

Click an employee's row in the table and click "Edit" to edit their schedule for that date.

Edit Schedule

With this screen, you can edit schedules by selecting an option from each dropdown.
